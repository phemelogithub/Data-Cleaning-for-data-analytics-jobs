# Data-Cleaning-for-data-analytics-jobs
The goal of this project is to simulate real-world data cleaning work done by data analysts. Using a housing dataset, we identify and fix data quality issues to make the dataset ready for analysis and visualization.

### Key Skills Demonstrated

* SQL Data Cleaning

* Handling Missing and Null Values

* Data Transformation and Standardization

* Removing Duplicates

* Using String Functions for Address Parsing

* Writing Efficient SQL Queries

### Tools & Technologies

* MySQL Workbench (for running queries)

* SQL Server / PostgreSQL (queries are compatible)

* Excel / CSV Dataset (raw data input)
